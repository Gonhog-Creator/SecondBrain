# Statistical Process Control (SPC)

Real-time monitoring of process output using control charts to detect shifts and special causes.

## Key tools
- Xbar-R, Xbar-S, I-MR charts
- Control limits: ±3σ
- Western Electric / Nelson rules for special causes
