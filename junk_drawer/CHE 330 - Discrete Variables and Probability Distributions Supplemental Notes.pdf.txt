In Excel, there is a binomial distribution function called BINOMDIST(x, n, p, cumulative)
which can be used to calculate binomial probabilities. In the string specifications, x, n, and p
correspond to the number of successes, number of trials, and probability of success, respectively.
Permissible entry for cumulative is either true or false. If true is specified, BINOMDIST will
return the cumulative distribution function, which is the probability that there are at most x
successes. If false is specified, BINOMDIST will return the probability function, which is the
probability that there are x successes. In Example 5 above, the use of BINOMDIST(2, 6, 0.75,
false) gives a value of 0.032959, in exact agreement with the hand calculation.
